Step 1: Download & Open GitHub

If you haven't yet, download your config file below, then login to GitHub and open the NikGApps repository.

Open NikGApps Repo →

Step 2: Fork the Repo

Click the 'Fork' button in the top-right corner of the NikGApps/Config repository page to create a copy in your own GitHub account.

Step 3: Upload Config

In your new forked repository, open the folder for your Android version. Click Add file > Upload files and upload your downloaded .config file.

Step 4: Commit Changes

Scroll down to the "Commit changes" box. Write a simple message like "Add my config" and click the green Commit changes button.

Step 5: Pull Request

Go back to the main NikGApps/Config repo, click the Pull requests tab, and click New pull request. Follow the prompts to submit it.

Final Step

Submit a Pull Request. The maintainers will review your config! 🎉

After it's merged, you can find your zip on SourceForge.

How to make a Token

  1. Create an account on GitHub if you haven't already.
  2. Go to your GitHub Personal Access Tokens (classic) page.
  3. Click on Generate new token -> Generate new token (classic).
  4. In the Note field, type something like "NikGApps Config".
  5. Under Select scopes, check the box next to public_repo. (Required to fork the repository and upload your file).
  6. Scroll to the bottom and click Generate token.
  7. Copy the generated token (starts with `ghp_`). You won't be able to see it again!
  8. Paste the token into the text field here.
